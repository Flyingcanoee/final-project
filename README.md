# Описание
Веб приложение, которое отображает список покемонов и позволяет их ловить и отпускать. 
При поимке покемона, его данные сохраняются в local storage. 
Использованные технологии: React, Bootstrap, Sass

# Установка и запуск
```
npm install
npm start
```
